# 🧞‍♂️ Project Aladdin 2.0 — 360° Market Intelligence

**Aladdin** — это мощная система маркетинговой разведки для рынка российской недвижимости. Инструмент автоматически агрегирует данные из 10 различных источников, оценивает спрос, инфраструктуру, собирает тональность отзывов (sentiment analysis), сканирует социальные сети и при помощи ИИ создает готовые портреты покупателей.

## ✨ Ключевые возможности

- **Анализ спроса (Wordstat 3.0)**: Классификация намерений пользователей (коммерческий/проблемный спрос), отсеивание шума через проприетарный Oracle Score.
- **Интеллектуальный поиск инфраструктуры**: Оценивает близость критически важных объектов (школы, больницы, магазины) с учетом контекста («в 500 метрах», «в шаговой доступности»).
- **Анализ тональности (Real Voice)**: Поиск реальных отзывов и жалоб дольщиков, расчет индекса лояльности.
- **Радар Социальных Сетей**: Автоматический сбор Telegram-чатов дольщиков, VK-сообществ и обзоров на YouTube.
- **Генерация Персон (Google Gemini)**: На основе собранных болей и цифр, ИИ создает готовые психологические портреты покупателей и стратегию продаж.
- **Авто-Конфигурация**: Система сама определяет город и бренд застройщика из поискового запроса.

## 🚀 Быстрый старт

### 1. Настройка окружения
В корневой папке проекта создайте файл `.env` и добавьте туда API-ключи:

```env
# Обязательные ключи (для работы ядра):
YANDEX_API_KEY=ваш_ключ
YANDEX_FOLDER_ID=ваш_folder_id

# Дополнительные ключи (расширяют функционал):
GOOGLE_API_KEY=ваш_ключ_gemini   # Для ИИ-генерации портретов покупателей
VK_TOKEN=ваш_сервисный_токен     # Для глубокого сканирования ВКонтакте
JINA_API_KEY=ваш_jina_ключ       # (Опционально) увеличивает лимит парсинга веб-страниц
```

### 2. Запуск полного 360° сканирования
Просто передайте системе фразу, которую хотите проанализировать. 
Формат: `[Тип недвижимости/Название ЖК] [Город]`

```bash
python scripts/aladdin.py "жк светский лес сочи"
python scripts/aladdin.py "таунхаус краснодар"
```

Отчет автоматически сохранится в формате JSON в папке с проектом, а в консоль будет выведена сжатая бизнес-аналитика.

## 📊 Режимы работы

### Режим 1: Полный пайплайн (Aladdin 360)
Запускает все 10 модулей друг за другом. Идеально для подготовки к запуску нового объекта или анализа конкурента.
👉 `python scripts/aladdin.py "ваш запрос"`

### Режим 2: Точечная работа с Wordstat CLI
Если вам нужны только данные по спросу, регионам или трендам, используйте точечный модуль:

```bash
# Классификация намерений (Интент анализ)
python scripts/yandex_wordstat.py "жк флора сочи" --method getTop --intent-analysis

# Выявление самых горячих регионов (с фильтром Oracle Score)
python scripts/yandex_wordstat.py "купить квартиру в сочи" --method getRegions --oracle-filter
```

## 🛠 Настройка (Для продвинутых)

Файл `scripts/config.py` управляет всей системой. Если вы хотите проанализировать неизвестный ЖК или коттеджный поселок, чтобы система не искала его по всей стране, вы можете добавить его в словарь известных проектов внутри `config.py`:

```python
_KNOWN_PROJECTS = {
    "светский лес": {
        "city": "Сочи",
        "district": "мкр. Бытха, р-н Хостинский",
        "brand": "ТОЧНО",
    },
    # Добавьте свой проект сюда...
}
```

## 🧠 Как писать правильные запросы
1. **Для оценки конкретного ЖК**: `"жк [Название] [Город]"` — точный срез.
2. **Для оценки ниши**: `"купить [тип недвижимости] в [город]"` — выявляет коммерческий спрос.
3. **Для поиска негатива конкурента**: `"[застройщик] [город] обман"` — собирает чужие боли для ваших продаж.
